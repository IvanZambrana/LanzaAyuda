AYUDA INICIO


AppAcademia es una aplicación de gestión académica donde tendremos la posibilidad de crear cursos en los que podremos insertar alumnos mediante la realización de matrículas.

Esta aplicación está ligada a una base de datos donde se almacenarán todos los registros realizados.


Este manual está elaborado con el fin de poder brindar al usuario final un conocimiento adecuado de la aplicación así como facilitar la navegación por las diferentes pantallas de la misma. También incluye pautas de uso de la propia ayuda.

Por otro lado, este manual estará dotado de diferentes apartados que incluirán capturas de pantalla para una mejor comprensión del mismo.

Para mejor manejo del manual de usuario, se recomienda el uso del Índice proporcionado en la página anterior.
